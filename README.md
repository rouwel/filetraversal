Hello There. 
Have you ever tried to find a file or folder  that was so well hidden on your C drive or root directory that you just gave up, well then this is the repo for you, to help get this program up to speed follow the instruction below to contribute.
This repository holds a skeleton of a program that should be able to sift through multiple folders  in a drive and locate a specific folder or file.
Right now all it does is identifies folders in the same directory it is in.
To see for yourself 
<pre>
clone https://github.com/rouwel/filetraversal.git
</pre>

Then proceed to add a folder of your choosing into the same folder you have cloned.
After this run the command
<pre>
go run . "foldername"
</pre>
Output shoud be 
<pre>
Directory Found
</pre>
